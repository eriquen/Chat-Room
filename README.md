# Chat-Room
Chat Room using HTML5, CSS, and JQuery. Use Socket.io to connect to the Express.js server.

### Requirement 
Node.js<br>
Express.js<br>
Socket.io<br>

### Install all dependencies

```
E:\chat room> npm install
```

### Run server

```
E:\chat room> node server.js
```

### Open client

```
http://localhost:8787
```
